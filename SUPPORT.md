# Support

URP is a community-maintained, pre-1.0 open-source project.

## Ask for help

1. Search the [documentation](https://thewisecrab.github.io/urp/) and existing
   [issues](https://github.com/thewisecrab/urp/issues).
2. Use a GitHub Discussion for architecture, adoption, and usage questions once
   Discussions are enabled.
3. Open a structured bug report for reproducible defects.

Include the URP version or commit, operating system, Python version, deployment
mode, exact command, redacted configuration, expected behavior, actual behavior,
and a minimal reproduction. Do not include credentials, payloads, raw prompts, or
unredacted manifests.

## Security

Do not open public issues for vulnerabilities. Follow [SECURITY.md](SECURITY.md)
and use GitHub private vulnerability reporting.

## Commercial support

No commercial support or service-level agreement is bundled with the Apache-2.0
project. Organizations may build their own support arrangements without changing
the open-source license.
