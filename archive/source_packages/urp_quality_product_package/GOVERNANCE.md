# Governance

URP uses maintainer-led open governance with public RFCs for major changes.

## Roles

- Core maintainer
- Area maintainer
- Security maintainer
- Reviewer
- Plugin maintainer

## Major changes require RFCs

Examples:

- manifest schema change;
- policy semantics change;
- plugin ABI change;
- new default reducer;
- security model change.

## Compatibility

Stable specs must support migration and deprecation windows.
